Imtahanlar

Imtahan nəticələri ilə bolüşmək / Imtahana nəzarətçi təyin etmək və s.

Grupların idarə edilməsi

Yeni qrup əlavə etmək / Tələbə əlavə etmək / Müəllim təyin etmək / imtahan vaxtını təyin etmək və s.

Tələbələr

Yeni tələbə akkauntu yaratmaq / akkauntu silmək

Müəllimlər

Yeni müəllim akkauntu yaratmaq / akkauntu silmək

Elanlar

Yeni elan yaratmaq / Elan silmək / Elanda dəyişiklik etmək

Cədvəl

Leksiya və seminarların cədvəlini qurmaq / İmtahanlarin cədvəlini qurmaq
